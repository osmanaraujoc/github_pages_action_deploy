¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web.

¿Cómo se crea una lista en HTML?

Para crear una lista en HTML, puedes utilizar las etiquetas <ul> (lista desordenada) o <ol> (lista ordenada) para definir la lista, y luego <li> para cada elemento de la lista.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para dar formato y diseño a las páginas web.
